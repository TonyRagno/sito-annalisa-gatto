# Sito Annalisa Gatto

Questo progetto e stato ripensato come sito statico semplice da gestire.

## Come aprirlo

Metodo piu semplice:

- fai doppio clic su `index.html`

Oppure:

- apri `index.html` dal browser

Non servono `npm`, build o dipendenze.

## File principali

- `index.html`
  Home page del sito.
- `privacy.html`
  Pagina privacy.
- `cookie-policy.html`
  Pagina cookie.
- `styles.css`
  Tutta la grafica e il layout.
- `script.js`
  Solo interazioni leggere: menu mobile, animazioni in ingresso, anno automatico.
- `Annalisa.jpg`
  Foto profilo usata nel sito.

## Come modificarlo

Se vuoi lavorare come in un sito HTML classico:

- modifica testi e struttura in `index.html`
- modifica colori, spaziature e grafica in `styles.css`
- modifica il comportamento del menu in `script.js`

## Contatti

I pulsanti principali non usano form complessi.
Aprono direttamente una email precompilata verso:

- `psicologannalisagatto@gmail.com`

Questa scelta e intenzionale:

- meno attrito per il primo contatto
- nessuna falsa area prenotazione
- nessuna raccolta superflua di dati sensibili sul sito
